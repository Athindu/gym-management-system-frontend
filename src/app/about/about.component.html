Gym Management System



  • Westminster Gym Management System is a web application which records all the data regarding the members of a certain fitness centre. Members can be either studetns, senior citizens or default members.
  • Features of the system,
      •    List of all the members
         Search members according to few parameters
         Manager can add, view, delete, sort, save to a file the details of the system
  • Developed using Spring, Angular and Mongo DB.
